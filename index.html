What is uNexus

uNexus is a gaming-first Linux shell built around speed, focus, and a polished desktop experience. Currently running natively on Arch Linux + Hyprland on real hardware. Built with Qt6/QML and C++, it replaces the entire desktop layer with a unified shell purpose-built for gamers — no bloat, no compromises.

"Make Linux gaming feel immediate, focused, and polished out of the box."

What is uNexus

Linux gaming,
reimagined.

uNexus is a gaming-first Linux shell built around speed, focus, and a polished desktop experience. Currently running natively on Arch Linux + Hyprland on real hardware.

Built with Qt6/QML and C++, it replaces the entire desktop layer with a unified shell purpose-built for gamers — with native Steam, Lutris, Heroic and Flatpak integration.

"Make Linux gaming feel immediate, focused, and polished out of the box."

Capabilities

Built for gamers,
engineered for Linux.

Every feature designed with performance and usability in mind.

🎮
Game Mode Integration
Top bar with clock, network, battery and Game Mode toggle. MangoHud and GameMode native support for maximum performance.
🚀
Real App Launcher
App launcher with search and categories. Real app launch via C++ — no wrapper scripts or shell hacks.
🛒
Store Integration
Steam, Lutris, Heroic and Bottles integration out of the box. Flatpak fallback for any gaming app not natively available.
📊
Performance Overlay
CPU/GPU/RAM stats overlay with real-time monitoring. System and Gaming side docks for quick access.
🔒
Custom Login Screen
Beautiful login screen with avatar, animated clock and password input. Plymouth boot splash for a seamless startup experience.
⚙️
Settings Panels
uNexus Settings panel and dedicated Game Settings panel. Full PT-BR and English language support built in.
🗂️
uNexus Files
Built-in file manager designed for the gaming desktop workflow. Notification system for background events.
🌐
Wayland Native
Built from the ground up for Hyprland/Wayland. Vulkan and OpenGL rendering with auto-start via Hyprland config.

Done

Implement system-wide keybindings for launcher, Game Mode toggle, screenshots and quick settings.
In Progress

Replace placeholder install commands with real Flatpak invocations. Handle install, update, remove and launch from within the shell.
In Progress

Planned

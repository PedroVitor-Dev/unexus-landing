01 / Origin Story

Linux
gaming,
reimagined.

It started with a question: why does gaming on Linux still feel like an afterthought? The hardware is there. The games are there. The shell just wasn't.

uNexus is a gaming-first Linux shell running natively on Arch Linux + Hyprland on real hardware — built with Qt6/QML and C++ to replace the entire desktop layer with one purpose-built for players.

"Make Linux gaming feel immediate, focused, and polished out of the box."
$ neofetch --config none OS: uNexus OS (Arch Linux) WM: Hyprland (Wayland) Shell: Qt6/QML + C++ GPU: Vulkan / OpenGL License: GPL-3.0 $ uNexus --game-mode on Game Mode: ACTIVE ✓ MangoHud: ENABLED ✓ Flatpak: READY ✓ $

03 / What it does

Built for
gamers.
Engineered
for Linux.

Every feature is deliberate — no settings that don't matter, no panels you'll never open.

🎮
Game Mode Integration
Top bar with clock, network, battery and Game Mode toggle. MangoHud and GameMode native support for maximum performance.
🚀
Real App Launcher
Search and categories. Real app launch via C++ — no wrapper scripts, no shell hacks, no delay.
🛒
Store Integration
Steam, Lutris, Heroic and Bottles. Flatpak fallback for anything not natively available.
📊
Performance Overlay
Real-time CPU/GPU/RAM monitoring. System and Gaming side docks for instant access.
🔒
Custom Login Screen
Avatar, animated clock, password. Plymouth boot splash for a seamless startup sequence.
⚙️
Settings Panels
uNexus Settings and Game Settings panels. PT-BR and English built in from day one.
🗂️
uNexus Files
Built-in file manager for the gaming desktop. Notification system for background events.
🌐
Wayland Native
Ground-up Hyprland/Wayland support. Vulkan and OpenGL rendering out of the box.
